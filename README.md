# Student Record Management System
 
A command-line application built in Java that manages student records using a Binary Search Tree (BST).
 
## Features
 
- Add student records with Name, Department, and CGPA
- Auto-generate unique roll numbers on registration
- Search records efficiently by roll number via BST
- Prevent duplicate roll number entries
## Tech Stack
 
- **Language:** Java
- **Build Tool:** Maven
- **Data Structure:** Binary Search Tree (BST)
## Prerequisites
 
- Java JDK 8 or higher
- Apache Maven 3.6 or higher
